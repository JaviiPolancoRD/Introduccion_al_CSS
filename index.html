ACTIVIDAD 1 CSS (Valor 10 puntos).

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación visual de un documento HTML o XML. Permite definir cómo se ven los elementos en una página web.

2. ¿Cuál es la función principal de CSS?

La función principal de CSS es separar el contenido estructural (HTML) de la presentación visual (estilos). Esto permite tener un mayor control sobre el diseño y la apariencia de una página web.

3. ¿Cómo se puede incluir CSS en un documento HTML?

Hay varias formas de incluir CSS en un documento HTML: Usando etiquetas style dentro del head del documento HTML. Enlazando un archivo CSS externo usando la etiqueta link dentro del head del documento HTML.

4. ¿Qué son los selectores en CSS?

Los selectores son patrones que se utilizan para seleccionar los elementos HTML a los que se les aplicarán estilos. Pueden ser selectores de tipo, de clase, de ID, de atributo, entre otros.

5. ¿Qué son las propiedades y valores en CSS?

Las propiedades son atributos que definen el aspecto de los elementos seleccionados, como el color, tamaño, márgenes, entre otros. Los valores son los ajustes específicos que se aplican a esas propiedades, por ejemplo, color: blue;.

6. ¿Cómo se aplica el modelo de caja en CSS?

El modelo de caja en CSS describe cómo se calculan el tamaño total de un elemento y cómo se distribuyen sus contenidos y márgenes. Incluye el contenido, el padding, el borde y el margen de un elemento.

7. ¿Qué es el "cascading" en CSS?

El "cascading" se refiere a la manera en que CSS resuelve los conflictos cuando múltiples estilos se aplican a un mismo elemento. Utiliza reglas de precedencia y especificidad para determinar cuál estilo prevalece.

8. ¿Qué son las media queries?

Las media queries son reglas de CSS que permiten aplicar estilos basados en las características del dispositivo donde se está visualizando la página, como el ancho de la pantalla, la resolución, orientación, etc. Se utilizan principalmente para crear diseños responsivos.

9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

Algunos estilos comunes incluyen ajustes de tipografía (como tamaño y tipo de fuente), colores de fondo y texto, márgenes y paddings para espaciado, y estilos de borde y sombra para dar profundidad visual a los elementos.

10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

Para crear un diseño responsivo con CSS, se utilizan principalmente media queries para ajustar el diseño y los estilos según el tamaño y características del dispositivo. Esto puede incluir la reorganización de elementos, cambios en tamaños de fuente y ajustes en márgenes y paddings para mejorar la experiencia de usuario en distintos dispositivos.
